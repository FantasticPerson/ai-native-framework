# Playground

输入一句指令，看 AI 如何在**能力清单白名单**里编排操作——产出操作计划，并高亮命中的能力。

<Playground />

## 它演示了什么

- **读结构，不猜屏幕**：右侧的能力清单是构建时从代码结构提取的。LLM 只能在这个白名单里编排。
- **精确编排**：一句自然语言 → 一串确定的 `navigate` / `fill` / `click` 步骤，每步对应清单里的一个能力。
- **白名单约束**：清单外的模块或操作会被 `parsePlan` 拒绝，不会执行。

> 本页是静态演示（预置 manifest + mock provider，不接真实 LLM）。真实接入见[快速开始](/guide/getting-started)。
