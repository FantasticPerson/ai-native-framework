features:
  - title: Read structure, don't guess the screen
    details: At build time an AST extracts a "capability manifest" from the routes, forms and buttons already in your code. The LLM orchestrates within a compile-time whitelist instead of guessing visually at runtime.
  - title: Visible & controllable
    details: A virtual cursor moves, values are typed character by character, targets are highlighted. Users understand it, trust it, and can interrupt at any time. Not a black box.
  - title: Access spectrum
    details: Auto-inference (zero changes) → config gap-filling → manual annotation. Low floor + high ceiling, not either-or.
  - title: Framework-agnostic core
    details: The core knows no frontend framework. React and Vue adapters coexist and run the same core — the counter-proof that the core is truly decoupled.
